# autobg

一个用于生成与图片宽高比相匹配的 CSS 样式代码的工具，参考自 [littlee/autobg.macro](https://github.com/littlee/autobg.macro) 项目，现已支持 UnoCSS 版本。

## 功能特点

- 🎯 支持 Babel Macro 和 UnoCSS 两种使用方式
- 📐 自动计算并生成背景图片的宽高比样式
- 🛠️ 支持 webpack 和 vite 构建工具
- 🔗 支持路径别名配置
- 📍 支持相对路径和绝对路径
- ⚙️ 支持自定义单位和设计稿宽度
